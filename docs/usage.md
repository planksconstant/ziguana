### Usage (Testing Purposes) can be removed in final shipment as it would be included in Readme.md 
* run ```sudo zig build install --prefix /usr/local``` give sudo permissions 
* Wait for the build to complete

* run ```ziguana testCode/<FileName>.zg --tokens``` to view the token stream

* run ```ziguana testCode/<FileName>.zg --astprint``` to view the Abstract Syntax Tree

* run ```ziguana testCode/<FileName>.zg --check``` to view the errors in the source code
